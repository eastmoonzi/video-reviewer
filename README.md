# 视频审查工作台

一站式视频模型输出审查平台，支持对模型生成内容进行可视化审查和打分标注。

## 功能特性

- **视频播放器**：支持倍速播放、精确跳转、区间循环
- **时间轴联动**：点击模型输出的时间段，视频自动跳转到对应位置
- **四维度审查**：
  - 时间段切分
  - 文本理解
  - 视觉理解  
  - 关键帧提取
- **多模型支持**：同一视频可对比多个模型输出，独立评分
- **快捷打分**：支持 1-3 分评分，快捷键操作
- **任务管理**：导入/导出、进度追踪
- **本地存储**：自动保存审查进度

## 快速开始

### 方式一：直接打开
双击 `index.html` 在浏览器中打开即可使用。

### 方式二：本地服务器（推荐）
```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx serve .
```
然后访问 http://localhost:8080

## 使用说明

### 导入任务
1. 点击左侧"导入"按钮
2. 选择 Excel 文件（.xlsx）
3. 文件格式见下方说明

### 审查流程
1. 从左侧任务列表选择任务
2. 观看视频，查看右侧模型输出
3. 点击时间段或关键帧可跳转到视频对应位置
4. 如有多个模型输出，点击顶部模型名称切换
5. 对四个维度分别打分（1-3分）
6. 添加备注（可选）
7. 点击"提交并下一条"

### 快捷键
| 按键 | 功能 |
|------|------|
| Space | 播放/暂停 |
| ← | 后退5秒 |
| → | 前进5秒 |
| 1-3 | 快速打分 |
| Tab | 切换维度标签 |
| Enter | 提交并下一条 |

## 导入格式（Excel）

### 文件结构

| 视频链接 | gemini3.0flash | claude-opus4.5 | gpt-4o | ... |
|---------|----------------|----------------|--------|-----|
| https://video1.mp4 | {"segment_detail":[...]} | {"segment_detail":[...]} | {"segment_detail":[...]} | ... |
| https://video2.mp4 | {"segment_detail":[...]} | {"segment_detail":[...]} | | |

- **第一行（表头）**：第一列可为"视频链接"或留空，后续列为模型名称
- **第二行起（数据）**：第一列为视频 URL（支持超链接），后续列为对应模型的 JSON 输出
- 同一视频可有多个模型输出（多列），每列独立评分

### JSON 数据格式

每个单元格的 JSON 结构：

```json
{
  "segment_detail": [
    {
      "time": [0, 30],
      "text": "这段视频展示了...",
      "vis": "画面中出现了...",
      "key_frame": [
        { "time": 5, "desc": "关键帧描述1" },
        { "time": 20, "desc": "关键帧描述2" }
      ]
    },
    {
      "time": [30, 60],
      "text": "接下来...",
      "vis": "视觉内容...",
      "key_frame": []
    }
  ]
}
```

| 字段 | 说明 |
|------|------|
| `time` | 时间段 `[开始秒数, 结束秒数]` |
| `text` | 文本理解/描述 |
| `vis` | 视觉理解内容 |
| `key_frame` | 关键帧数组，每项含 `time`（秒）和 `desc`（描述） |

## 导出格式（Excel）

点击"导出"按钮，下载 `.xlsx` 文件，每行对应一个视频-模型组合：

| 任务ID | 视频URL | 模型名称 | 状态 | 时间段切分 | 文本理解 | 视觉理解 | 关键帧提取 | 时间段备注 | 文本备注 | 视觉备注 | 关键帧备注 |
|--------|---------|---------|------|-----------|---------|---------|-----------|-----------|---------|---------|-----------|
| task-1 | https://... | gemini3.0flash | 已完成 | 3 | 2 | 3 | 2 | | 文本有误 | | |
| task-1 | https://... | claude-opus4.5 | 已完成 | 2 | 3 | 2 | 3 | 切分不准 | | | |
| task-2 | https://... | gemini3.0flash | 未完成 | | | | | | | | |

- **评分范围**：1-3 分
- **四个评分维度**：时间段切分、文本理解、视觉理解、关键帧提取
- **备注**：每个维度可单独添加备注
